Firebase設定 QRコード生成

期限管理アプリ用 — PCで生成してiPhoneでスキャン

📋 手順
1
Firebaseコンソール → プロジェクト設定 → マイアプリ → 「構成」を選択
💡
このページはスマホからも開けます。アプリの ⚙️ → 「QRコードを生成する」 からアクセスできます。
2
const firebaseConfig = { … } のコードをすべてコピー
3
下のテキストエリアに貼り付けて「QRコードを生成」をクリック
4
表示されたQRコードをiPhoneの期限管理アプリでスキャン → 自動設定完了！
✅ QRコードが生成されました
このQRコードをiPhoneのカメラまたは
アプリの設定画面からスキャンしてください
📱 アプリでのスキャン方法：
期限管理アプリ → 右上 ⚙️ → 「QRコードでスキャン」 タップ
→ カメラにこのQRコードを向けると自動で設定が完了します
📱 アプリに戻る
このQRコードには Firebase の設定情報が含まれています。
URLや画像を外部に公開しないでください。
